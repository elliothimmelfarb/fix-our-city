# Fix Our City
### Crowd-Sourced City Maintenance Reporting App

### [www.FixOur.City](https://www.fixour.city/)



### Built Using:
- React and Redux
- Google Maps api ([google-map-react](https://github.com/istarkov/google-map-react))
- Express
- MongoDB Geospatial Services
- Paired Programming



### Features
- Automatic location discovery
- Upvote and downvote issues
- Mark issues as fixed
- Mobile friendly (take pictures from your phone in-browser)
